TableSorter with frequency sorting

Usage:

  • Init plugin for table jQuery('.table-sorter').frequencyTableSorter();
  • If need to use frequency sort type in one of the columns by default - just add className "sorter-frequencySort" on the <th>
  • The connection between the values in the select and the columns, for example, are implemented through:
    <option value="email">Email</option> => <th data-frequency="email">Email</th>.
    It can be changed to any other convenient option, it does not affect the work of the plugin in any way.
